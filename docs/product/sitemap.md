# CrowdFlow Platform Sitemap

This document outlines the core screens and views for the CrowdFlow platform, categorized by user persona. Use these descriptions for generating high-fidelity screens.

---

## 1. Public & Attendee Facing
These screens use a **Modern, Approachable, and Seamless** aesthetic.

### [Screen] Landing Page (`/`)
- **Purpose**: Brand introduction and portal entry.
- **Key Elements**: 
  - Hero section with kinetic typography.
  - "Live Venue Pulse" overview (simplified heatmap).
  - Quick-start for ticket holders.
- **Aesthetic**: Vibrant accents, smooth transitions, premium typography (Outfit).

### [Screen] Attendee Dashboard (`/dashboard`)
- **Purpose**: Personal command center for the visitor.
- **Key Elements**:
  - Personal Queue status (wait times).
  - "Smart Wayfinding" map view.
  - AI Assistant Chat (BhashaLens integration).
  - Flash Alerts (Safety/General).
- **Aesthetic**: Personal, clean, high-contrast readability.

### [Screen] AI Assistant Interface
- **Purpose**: Deep-dive interaction for directions and info.
- **Key Elements**:
  - Persistent chat bubble.
  - Multi-modal responses (Text + Direction Cards).
  - Quick-action buttons (e.g., "Find Food", "Nearest Exit").

---

## 2. Operations & Administrative
These screens use the **"Sentinel Control"** aesthetic: Dark mode, glassmorphism, high-density data, and kinetic interactions.

### [Screen] Operations Console (`/operations`)
- **Purpose**: Main hub for venue oversight.
- **Key Elements**:
  - **Live Heatmap**: SVG-based venue overview with real-time zone pulses.
  - **Zone Intelligence**: Collapsible sidebar with telemetry for every sector.
  - **Global Stats**: Saturation, Throughput, and Attendee Count.
- **Aesthetic**: Cyber-Ops premium, deep navies/blacks, electric cyan (`#14ffd8`) accents.

### [Screen] Incident Command (Alert Center)
- **Purpose**: Real-time management of security and safety issues.
- **Key Elements**:
  - Live Alert Stream with priority filtering.
  - Actionable items (Assign Staff, Resolve).
  - Alert Detail Modal with historical trends for the location.
- **Aesthetic**: High-urgency pulsars, data-dense logs.

### [Screen] Staff Coordination (Portal)
- **Purpose**: Deployment and communication tool for ground units.
- **Key Elements**:
  - "Assigned to Me" list of tasks.
  - Zone location lock (Geo-sync).
  - Comms link indicator.
- **Aesthetic**: Functional, high-visibility actions for field usage.

### [Screen] System Health & Monitoring
- **Purpose**: Technical oversight of IoT and API integrity.
- **Key Elements**:
  - SSE Stream health indicators.
  - Latency/Latency trend graphs.
  - Global system resolve status.
- **Aesthetic**: Technical, grid-based, monochromatic with status pulses.
